Memento-Skills is a self-evolving agent framework built around skills as first-class units of capability. Instead of treating tools as a flat list, it treats them as a library the agent can retrieve, execute, evaluate, repair, and rewrite over time.

The core loop is simple: Read - Execute - Reflect - Write. When a task fails, the system does not just retry. It tries to identify the weak skill, improve it, and write the improved capability back into memory.
